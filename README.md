# Customer Churn Analysis

This project explores customer churn through a complete early-stage data analytics workflow. It includes data gathering, cleaning, exploratory data analysis, and preparation of a modeling-ready dataset to support future churn prediction work.

## Project Overview

The objective of this project is to better understand customer churn by working through the main stages of a real-world analytics process.

In this notebook, I:

- gathered data from multiple sources
- cleaned and standardized the datasets
- explored customer behavior through analysis and visualization
- created new variables relevant to churn analysis
- prepared a structured dataset for future predictive modeling

This project focuses on the analytical groundwork required before building a machine learning model.

## Project Scope

The business goal is to identify patterns related to customer cancellations and prepare data that can later be used to predict which customers are more likely to churn.
